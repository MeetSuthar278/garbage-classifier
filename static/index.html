🔍 Explainability — Grad-CAM

Gradient-weighted Class Activation Mapping (Grad-CAM) hooks into the final convolutional block of EfficientNet-B0 and computes the gradient of the predicted class score with respect to the feature maps. Regions with high gradient magnitude are highlighted in red/yellow — showing where the model looked to make its decision. This makes the model interpretable and verifiable, which is critical for real-world deployment of waste classification systems.

⚠️ Limitations & Future Work

  • The BERT branch uses image filenames as text — in real deployment a caption or OCR on packaging label would be more informative.
  • The model is trained on a balanced dataset; real-world class distributions may differ significantly.
  • Very small or heavily occluded items may confuse the classifier.
  • Future work: real-time webcam stream, mobile-optimised MobileNetV3 branch, caption-based text input via BLIP-2.
